### Продукт А — Монитор торгов → Telegram
**Что автоматизирует:** ежедневный ручной мониторинг torgi.gov.ru и ГИС Торги
**Стек:** n8n + Telegram Bot API (RSS torgi.gov.ru)
**Время на запуск:** 1 день
**Уже частично в плане:** Этап 10

- [ ] Настроить фильтры под свои регионы и ВРИ
- [ ] Получать уведомление сразу при появлении нового лота
- [ ] Добавить фильтр по начальной цене (только выгодные лоты)

---

### Продукт Б — AI-анализатор ПЗЗ и Генплана
**Что автоматизирует:** час ручного чтения градостроительных документов
**Стек:** Claude API (загрузка PDF → вопрос → ответ с цитатой)
**Время на запуск:** 2–3 дня

- [ ] Загружаешь PDF любого ПЗЗ → спрашиваешь конкретно: «можно ли ИЖС в зоне Ж-2?»
- [ ] Агент отвечает за 30 секунд вместо часа чтения
- [ ] Работает с Генпланами, ПЗЗ, ответами администраций, отказами

---

### Продукт В — Трекер участков (личная CRM)
**Что автоматизирует:** хаос в голове когда ведёшь несколько участков одновременно
**Стек:** Notion (бесплатно, без кода) или Lovable + Supabase
**Время на запуск:** несколько часов (Notion) или 2–3 дня (веб-приложение)

- [ ] Карточка на каждый участок: стратегия, статус, дата подачи, дедлайн
- [ ] Счётчик заявлений на муниципалитет (лимит 3–5 по технике безопасности)
- [ ] Напоминание о 3-летнем сроке освоения (новый закон с 2025)
- [ ] Видно одним взглядом: что ждёт ответа, что одобрено, что просрочено

---

### Продукт Г — Калькулятор стоимости выкупа
**Что автоматизирует:** ручной расчёт выкупной стоимости по кадастровой стоимости и ставке региона
**Стек:** Lovable.dev (только фронтенд, без сервера)
**Время на запуск:** 1 день

- [ ] Ввод: регион, ВРИ, кадастровая стоимость
- [ ] Вывод: стоимость выкупа в собственность, ставка аренды, итоговая сумма
- [ ] Данные: файл «Стоимость выкупа земельных участков.pdf» из базы знаний

---

### Продукт Д — Помощник для схемы СРЗУ (текстовые инструкции)
**Что автоматизирует:** подготовку к работе в НСПД и АРГО
**Стек:** Claude API (чат + база знаний)
**Время на запуск:** добавить в Штурман как отдельный раздел

- [ ] Пошаговая инструкция: какие кнопки нажать в НСПД для конкретного участка

- [ ] AI проверяет: не нарушает ли схема границы охранных зон по описанию
- [ ] Объяснение на что обратить внимание перед отправкой схемы

**Что автоматизирует:** первичный отбор клиентов и прогрев к консультации
**Стек:** Typeform / Marquiz или роутер Штурмана как отдельная страница
**Время на запуск:** 2–3 часа
